# How to run

1. Download Repository
2. Inside directory:

    Linux
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    ```
    
    Windows
    ```powershell
    python3 -m venv .venv
    .venv/bin/Activate.ps1
    pip install -r requirements.txt
    ```
3. Open the ```.ipynb``` file and run the notebook
